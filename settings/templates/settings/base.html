    Über uns

    🤖🔍📷 Machine-Learning-Modell 🌟🚀 .Mit diesem Machine-Learning-Modell können Sie Ihr medizinisches Bild hochladen und Vorhersagen erhalten. 📷🔍💡 Das Modell wurde mit Teachable Machine erstellt.

    So funktioniert es:

    • Laden Sie ein Bild hoch.
    • Das Modell analysiert das Bild und gibt eine Vorhersage aus.
    • Die Vorhersage wird als Prozentsatz und Klassifizierung angezeigt.

    Probieren Sie es aus und entdecken Sie die Möglichkeiten der maschinellen Bilderkennung! 🚀💭🔬
